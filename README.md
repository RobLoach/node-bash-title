# node-bash-title

A NodeJS library to change the title of your BASH shell. Usefull in development: name your bash screens and display error or info messages. You can use even uni-code üçª

## Install

```
  npm install node-bash-title --save
```

or 

```
  yarn add node-bash-title
````


## Usage 

In shell script

```
  const setTitle = require('node-bash-title');
  setTitle('üçª  Server');
```
